# Time Manager API

## О проекте

`Time Manager API` — это учебный backend-проект на FastAPI для лабораторной работы по серверной разработке.

Основная идея проекта: дать пользователю простой API для управления своим временем через задачи, категории, теги и напоминания.

Проект поддерживает:

- регистрацию и вход;
- JWT-аутентификацию;
- работу с задачами;
- учет статуса и приоритета задач;
- разбиение задач по категориям;
- маркировку задач тегами;
- создание напоминаний;
- миграции через Alembic.

## Почему это именно тайм-менеджер

В центре предметной области находится сущность задачи:

- задача имеет дедлайн;
- задача имеет приоритет;
- задача имеет статус выполнения;
- задача хранит планируемое и фактическое время;
- задача может быть включена в категорию;
- задача может иметь набор тегов;
- задача может иметь несколько напоминаний.

То есть система помогает не просто хранить список дел, а структурировать работу по времени и контексту.

## Ключевые особенности реализации

- конфигурация БД берется из переменных окружения;
- строка подключения для Alembic не хранится в `alembic.ini`;
- миграции создают схему базы данных;
- чувствительные данные пользователя не возвращаются наружу;
- код написан в учебном стиле и снабжен комментариями в нетривиальных местах.
